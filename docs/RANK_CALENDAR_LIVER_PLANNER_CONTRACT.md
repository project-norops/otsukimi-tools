# IRIAMランク管理カレンダーとライバー手帳の連携契約

## 目的

ライバー手帳は、IRIAMランク管理カレンダーの内容を**読み取り専用**で表示する。手帳からランク管理カレンダーの保存データを書き換えない。

## 保存データ

- 読み取り元: `sushiusa-tools:iriam-rank-calendar:v2`
- 手帳の保存先: `sushiusa-tools:liver-planner:v1`
- ランク管理カレンダーの保存形式・LocalStorageキーは変更しない。

## 手帳に反映する項目

ランク管理カレンダーと同じシミュレーションを使い、シミュレーション期間内の日付へ次を表示する。

1. 未入力日の標準予定 `IRIAM (+1)`
2. 手動で設定した `+1 / +2 / +4 / +6 / SKIP / 休み` とメモ
3. 週次・手動のスキップパス付与 `IRIAM（SP+N）`
4. ランクアップ、キープ、ランクダウン
5. デビュー記念日・日数記念日

表示範囲はランク管理カレンダーが保存している基準日とシミュレーション期間（現在は1〜3か月）に限定する。

## 変更時の必須確認

- `src/lib/liver-planner.test.ts` の連携テストを更新・実行する。
- 標準+1、SP+1、キープ、ダウン、記念日の少なくとも各1件を確認する。
- 手帳からランク管理LocalStorageへ書き戻しがないことを確認する。
- `npm.cmd test && npm.cmd run typecheck && npm.cmd run lint && npm.cmd run build` を成功させる。
